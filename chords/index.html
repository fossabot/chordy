Chords

Basics

A chord is an abstraction of a part of a tune or a riff. A song or a tune comprises of several chords played in varying rhythm. In chordy, chords can be created using the play function.

There are several broad ways to represent a chord. A chord can be represented as an array of numbers, where the numbers represent the fret positions of different strings in the chord. play understands this notation, and you can specify a chord to be played using an array of integers. In chordy, -1 denotes a string that is not played, and an open chord is 0. Other numbers denote the fret position of the string in the chord.

Alternatively, a chord can be specified by it’s chord family and type. The play function takes one required arugment, which can be a chord family like C or F#. There are a handful of notations to specify a chord family. For example, the C chord family can be represented as C, :C or "C". Similarly, F# is FSharp, :F! or "F!". A ! suffix denotes a sharp, and _ denotes a flat. Note that the string and symbol representations can be in lowercase as well.
